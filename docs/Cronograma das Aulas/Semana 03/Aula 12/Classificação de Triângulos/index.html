Classificação de Triângulos

Introdução​

Dados três valores X, Y, Z, verifiquem se eles podem ser os comprimentos dos lados de um triângulo e se forem escrever uma mensagem informando se é um triângulo eqüilátero, isósceles ou escaleno.

Observações:

  • O comprimento de um lado do triângulo é sempre menor do que a soma dos outros dois.

Classificações:

  • Eqüilátero: Todos os lados iguais
  • Isósceles: Dois lados iguais
  • Escaleno: Todos os lados diferentes

Passo a passo​

  • Entrada de Dados Básica
  • Verificação de Triângulo Válido
  • Classificação do Triângulo
  • Tratamento de Entradas Inválidas
  • Melhoras na legibilidade do código
  • Utilizar classe estática ClassificadorDeTriangulos

Versão 1 - Entrada de Dados Básica​

  • O sistema deve apresentar uma mensagem de boas-vindas
  • O sistema deve solicitar os três valores correspondentes aos lados do triângulo
  • O sistema deve exibir os valores digitados para confirmação
  • O sistema deve encerrar após mostrar os valores
static void Main(string[] args)
{
    while (true)
    {
        Console.Clear();
		Console.WriteLine("-------------------------------------------------");
        Console.WriteLine("Classificador de Triângulos");
        Console.WriteLine("-------------------------------------------------");
        
		Console.Write("Informe o valor do lado X: ");
		int ladoX = Convert.ToInt32(Console.ReadLine());

		Console.Write("Informe o valor do lado Y: ");
		int ladoY = Convert.ToInt32(Console.ReadLine());

		Console.Write("Informe o valor do lado Z: ");
		int ladoZ = Convert.ToInt32(Console.ReadLine());

        Console.WriteLine("-------------------------------------------------");
        Console.WriteLine("Valores informados:");
        Console.WriteLine($"Lado X: {ladoX}");
        Console.WriteLine($"Lado Y: {ladoY}");
        Console.WriteLine($"Lado Z: {ladoZ}");
        Console.WriteLine("-------------------------------------------------");

        Console.WriteLine("-------------------------------------------------");
        Console.Write("Deseja continuar? (s/N) ");

        string opcaoContinuar = Console.ReadLine()!.ToUpper();

        if (opcaoContinuar != "S")
            break;
    }
}

Versão 2 - Verificação de Triângulo Válido​

  • O sistema deve manter todos os requisitos da Versão 1
  • O sistema deve verificar se os três valores formam um triângulo válido
  • O sistema deve informar ao usuário se os valores formam ou não um triângulo válido
while (true)
{
    Console.Clear();
    Console.WriteLine("-------------------------------------------------");
    Console.WriteLine("Classificador de Triângulos");
    Console.WriteLine("-------------------------------------------------");

    
	Console.Write("Informe o valor do lado X: ");
	int ladoX = Convert.ToInt32(Console.ReadLine());

	Console.Write("Informe o valor do lado Y: ");
	int ladoY = Convert.ToInt32(Console.ReadLine());

	Console.Write("Informe o valor do lado Z: ");
	int ladoZ = Convert.ToInt32(Console.ReadLine());

    Console.WriteLine("-------------------------------------------------");
    Console.WriteLine("Valores informados:");
    Console.WriteLine($"Lado X: {ladoX}");
    Console.WriteLine($"Lado Y: {ladoY}");
    Console.WriteLine($"Lado Z: {ladoZ}");
    Console.WriteLine("-------------------------------------------------");

    bool medidasFormamTrianguloValido = 
        ladoX + ladoY > ladoZ &&
        ladoX + ladoZ > ladoY &&
        ladoY + ladoZ > ladoX;

    if (medidasFormamTrianguloValido)
        Console.WriteLine("O triângulo é valido.");
    else
        Console.WriteLine("Os valores informados não formam um triângulo válido.");

    Console.WriteLine("-------------------------------------------------");
    Console.Write("Deseja continuar? (s/N) ");

    string opcaoContinuar = Console.ReadLine()!.ToUpper();

    if (opcaoContinuar != "S")
        break;
}

Versão 3 - Classificação do Triângulo​

  • O sistema deve manter todos os requisitos da Versão 2
  • O sistema deve classificar o triângulo como Equilátero, Isósceles ou Escaleno, caso seja válido

{
    Console.Clear();

    Console.Write("Informe o valor do lado Y: ");
    int ladoY = Convert.ToInt32(Console.ReadLine());

    Console.Write("Informe o valor do lado Z: ");
    int ladoZ = Convert.ToInt32(Console.ReadLine());

    Console.WriteLine($"Lado Y: {ladoY}");

        ladoX + ladoY > ladoZ &&

public class ClassificadorDeTriangulos
{
    public static bool MedidasFormamTrianguloValido(int x, int y, int z)
    {
        return x + y > z && x + z > y && y + z > x;
    }

    public static string ClassificarTriangulo(int x, int y, int z)
    {
        if (x == y && y == z)
            return "Equilátero";
        else if (x != y && y != z && x != z)
            return "Escaleno";
        else
            return "Isósceles";
    }
}
