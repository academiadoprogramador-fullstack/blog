## Introdução

CSS (Cascading Style Sheets), ou **Folhas de Estilo em Cascata**, é uma linguagem usada para **estilizar elementos HTML** em páginas web. Com o CSS, você pode controlar:

- **Cores** (de texto, fundo, bordas etc.)
- **Fontes** (tipo, tamanho, estilo)
- **Espaçamento** (margens, preenchimento/padding)
- **Layout** (posição dos elementos, flexbox, grid)
- **Efeitos visuais** (sombreamento, transições, animações)

### Exemplo simples:

```html
<!DOCTYPE html>
<html>
<head>
  <style>
    h1 {
      color: blue;
      font-family: Arial;
    }

    p {
      background-color: lightgray;
      padding: 10px;
    }
  </style>
</head>
<body>
  <h1>Olá, mundo!</h1>
  <p>Este é um parágrafo estilizado com CSS.</p>
</body>
</html>
```

### Tipos de CSS:

1. **Inline** (no próprio elemento HTML):

   ```html
   <p style="color: red;">Texto em vermelho</p>
   ```
2. **Interno** (no `<style>` dentro do `<head>`)
3. **Externo** (em um arquivo `.css` separado)

CSS é essencial para deixar um site **bonito, responsivo e organizado**. Ele trabalha junto com HTML (estrutura) e JavaScript (interatividade).

      }

      }

        margin-top: 20px;
        margin-left: auto;

        padding: 5px 10px;

        color: black;
        background-color: #b084d1;

        border: 1px solid #b084d1;
        border-radius: 9px;
      }

      @@media (max-width: 768px) {

        main {
          padding: 20px;
          flex-direction: column;
          align-items: stretch;
        }

        header {
          padding: 20px;
        }

        main {
          align-items: center;
        }
      }
    </style>
  </head>
  <body>
    <nav>
      <h2>Gestão de Equipamentos</h2>

      <ul>
        <li><a href="/">Início</a></li>
        <li><a href="/fabricantes/visualizar">Fabricantes</a></li>
        <li><a href="/equipamentos/visualizar">Equipamentos</a></li>
